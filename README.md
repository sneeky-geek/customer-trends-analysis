# Customer Behavior Data Analysis
An end-to-end data analytics project analyzing customer shopping behavior using Python, SQL, and Power BI. This project demonstrates a complete workflow from data preparation to visualization and insight generation.


## Project Overview
This project demonstrates an end-to-end data analytics workflow for analyzing customer shopping behavior:

- **Data Preparation & Exploratory Data Analysis (Python)**: Clean and transform raw customer data
- **Data Analysis (SQL)**: Query data to extract insights on customer segments, loyalty, and purchase patterns
- **Visualization & Insights (Power BI)**: Build interactive dashboards highlighting key trends
- **Reporting**: Document findings and actionable business recommendations

## How to Use This Project

1. **Open Customer_Shopping_Behavior_Analysis.ipynb notebook**
   - Data Import
   - Data exploration
   - Data cleaning
   - Connection to SQL Database

2. **Load the data from Python notebook into MySQL/PostgreSQL/MS SQL Server**
   - Create a database in SQL
   - Run Python code to load data into SQL database
   - Open **customer_behavior_sql_queries.sql**
   - Answer Business Questions using SQL Queries

3. **Connect the SQL Database to Power BI**
   - Open **customer_behavior_dashboard.pbix**
   - Create interactive dashboard in Power BI

4. **Create Project Report and Presentation**
   - Document project findings
   - Build presentation deck
